# TradingView-like Chart (Final)

This final version includes:
- Manual column mapping UI for Date/Open/High/Low/Close/Volume
- Parsed preview (first rows) and mapped-sample preview
- Volume normalization toggle
- Candlestick chart with volume and SMA
- Instructions to run

Quick start:
1. npm install
2. npm run dev
3. Open the local URL (usually http://localhost:5173)
4. Upload your CSV, confirm headers and mapping, then Apply Mapping.
